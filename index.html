Sequência de Inicialização

Processo para Ligar

1. Ligue a mesa de som
2. Ligue a Régua de Energia
3. Ligue os amplificadores
4. Ligue o computador

Processo para Desligar

1. Desligue o Computador
2. Desligue os Amplificadores
3. Desligue a Régua de energia
4. Desligue a Mesa de som
Cerfifique-se que o computador está completamente desligado para somente depois desligar a régua.
